This plugin will automatically enroll in Moodle customers who buy the course in WooCommerce.


== Description ==

English.

This plugin keeps a list of all the product-course associations you define.
Then, when a customer buys the product, he or she is automatically enrolled in the associated course or courses.
A Moodle account is created first if the customer does not already have one.

You have the possibility to associate more than one product with a course, and you can have one product associated with multiple courses.

This plugin does not create products in WooCommerce. You are free to manage your products as you wish.


Français.

Cette extension inscrira automatiquement dans Moodle les clients qui achètent le cours dans WooCommerce.

Cette extension conserve une liste de toutes les associations de produit-cours que vous définissez.
Ensuite, lorsqu'un client achète le produit, il est automatiquement inscrit dans le ou les cours associés.
Un compte sur Moodle est créé en premier si le client n'en a pas déjà un.

Vous avez la possibilité d'associer plus d'un produit à un cours, et vous pouvez avoir un seul produit associé à plusieurs cours.

Ce plugin ne cré pas les produits dans WooCommerce. Vous êtes donc libre de gérer vos produits comme vous le souhaitez.


== Installation ==

Simple as any other plugin installation.

There is a configuration screen named Moodle in the Settings section of the WordPress Dashboard.
And you can manage the product-course associations in the Moodle option of the Products section.
